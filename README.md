특정 2가지 : safety- 노드간의 합의가 발생했다면, 어느 노드가 접근하든 그 값은 동일,

목표: 탈중앙 환경에서 모든 노드가 동일한 분산 원장을 가질 수 있도록 한다.

목표: 

1) 고유한 소유자와 자산을 식별

2) 허가된 사람만 그 자산에 접근하도록 보장
